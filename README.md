# Mimik Description

Mimik's collection of object meshes (collision + visual) and description format (URDF).

## Name convention
Convention used for all mesh names throughout this package:
- `p`&nbsp;&nbsp;   = primitive (single or collection)
- `bb`&nbsp;        = bounding box
- `cv`&nbsp;        = convex hull
- `24`&nbsp;        = 24 triangles
- `24k`             = 23,500 to 24,499 triangles
- `24m`             = 23,500,000 to 24,499,499 triangles
